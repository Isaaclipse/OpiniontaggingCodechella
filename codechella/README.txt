pre-trained model to classify text is from this github repo
- https://github.com/miguelfzafra/Latest-News-Classifier/tree/master/0.%20Latest%20News%20Classifier

The codes folder contains code to run a text-based demonstration of the feature
To run the text-based demonstration
- If you're entering tweet yourself: 
python main.py -w "enter tweet here" -ot [0 or 1]

- If you're reading tweet from a file
python main.py -r path_to_file

- If you're downloading live tweet for analysis
python main.py -dl "@userhandle"
